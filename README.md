# Laibah Qureshi's Attire 💎

A responsive e-commerce frontend for a premium jewelry brand. Built to showcase a luxury collection with a clean, product-first design.

### ✨ Live Preview
Open `index1.html` in any browser to view the site.

### 🛍️ Key Features
- **Premium Jewelry Collection** hero section 
- **3 Product Categories**: Luxury Necklaces, Signature Earrings, Elegant Bracelets
- **Product Cards** with images, names, and PKR pricing `Rs.`
- **Responsive Grid Layout** for desktop and mobile
